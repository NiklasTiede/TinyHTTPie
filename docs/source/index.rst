A Simple HTTP Client
====================

TinyHTTPie was created within the multi-part series "`The Evolution of a Script <https://the-coding-lab.com/posts/The-Evolution-of-a-Script/>`_".
It's purpose is to show the tooling one uses for more sophisticated python projects.

With tinyHTTPie you can do GET requests and print out header and body of the response.
